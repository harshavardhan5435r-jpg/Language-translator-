## What it does
- Translate text into any language

## Technologies Used
- Python
- Google Gemini AI
- Gradio
- python-dotenv

## How to Run

### 1. Clone the project
git clone https://github.com/harshavardhan5435r-jpg/Language-translator-.git

### 2. Install packages
pip install -r requirements.txt

### 3. Add API key
Create a .env file and add:
GEMINI_API_KEY=your_key_here

### 4. Run the app
python3.12 app.py
